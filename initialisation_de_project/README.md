# Project

## Contribution

Si vous souhaitez contribuer à ce projet, veuillez suivre ces étapes :
1. Faites un **fork** du projet.
2. Effectuez vos modifications dans votre fork.
3. Une fois terminé, soumettez une **Pull Request** (PR).
4. Assurez-vous que tous les labs qui ne sont pas dans l’organisation soient placés dans des repositories publics sur votre compte.

Les Pull Requests seront examinés et acceptés par **Yasmine** ou **Abdelhay**.
